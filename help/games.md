# `games` — Mini-Games

> **Category**: Player Control

---

## SYNOPSIS

```
games             # launch built-in mini-game player
```

## DESCRIPTION

Opens the built-in mini-games interface in standalone mode. This is a
separate application module (`wait_games.py`) that runs independently
from the main AI DJ session.

During PC mode, game injectors are automatically paused to avoid
interfering with continuous playback.
